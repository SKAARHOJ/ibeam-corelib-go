func Param ¶ 

func Param(pid uint32, did uint32, vals ...*pb.ParameterValue) *pb.Parameter

Param is the main function of the parameter helpers suite. It is used to create a pb.Parameter with a specified id and device. After that you can add as many parameter values as needed for the specific parameter (usually onbly one, more for parameters with dimensions that you want to update at the same time)
